Sobre nós:

Nosso objetivo implementar uma arquitetura serverless, que oferece benefícios significativos para redes e entrega de conteúdo, utilizando serviços da Amazon Web Services (AWS), como rede de entrega, baixa latência, armazenamento durável e segurança.

Esses benefícios fazem da arquitetura serverless uma opção atraente para empresas que procuram eficiência e escalabilidade.

  • Adicionar um registro de domínio externo (DNS), cadastrado e configurado no Amazon Route 53, para personalizar e assegurar o URL escolhido para o site de e-commerce. O Amazon Route 53 é um serviço web de DNS altamente disponível e escalável, que conecta as solicitações dos usuários a aplicações da Internet executadas na AWS ou em ambientes on-premises.
  • No aspecto de rede de entrega e baixa latência, implementamos uma distribuição no Amazon CloudFront, um serviço que distribui conteúdo com alta velocidade de transferência de dados e baixa latência, atendendo às solicitações através de uma rede global de pontos de presença.
  • Para assegurar a segurança dos clientes e a proteção dos recursos, redirecionamos todas as conexões HTTP para HTTPS e utilizamos o AWS Well-Architected Firewall (WAF) integrado ao Route 53, que permite provisionar e gerenciar facilmente.
  • Para o armazenamento de dados estáticos, o Amazon Simple Storage Service (S3), reconhecido por sua escalabilidade, disponibilidade de dados, segurança e desempenho de ponta.
  • Integrado o banco de dados não relacional DynamoDB com o Lambda, uma combinação ideal para otimização de custos. O Lambda se encaixa no modelo SaaS (Software as a Service), onde tudo é gerenciado pela plataforma. O principal benefício do Lambda é eliminar a preocupação com o provisionamento de aplicações ou serviços. O AWS Lambda é um serviço versátil e potente, que permite a empresas de qualquer tamanho executar código sem a necessidade de provisionamento e gerenciamento de servidores.
  • No intuito de aumentar a experiência do cliente/usuário, foram implementados dois serviços da AWS, Lex e Translate, com seus recursos inovadores facilitaram a interação do cliente/usuário a plataforma.
